## 1. 일반

### Q: Vibe Olympics가 무엇인가요?

**A:** Vibe Olympics는 AI와 함께 만든 디지털 창작물을 사고파는 마켓플레이스입니다.

- **판도라 샵**: 디지털 상품, AI 도서, 영상, 음악 구매/판매
- **교육 센터**: VIBE 코딩 학습 자료
- **커뮤니티**: 창작자들의 소통 공간

---

### Q: 회원가입은 무료인가요?

**A:** 네, 완전히 무료입니다. 가입 후 바로 구매와 판매를 시작할 수 있습니다.

---

### Q: VIBE 코딩이 무엇인가요?

**A:** VIBE 코딩은 AI 도구(ChatGPT, Claude 등)와 함께 코드를 작성하는 새로운 개발 방식입니다.

- 프로그래밍 경험이 없어도 시작 가능
- AI가 코드 작성을 도와줌
- 교육 센터에서 무료로 배울 수 있음

---

### Q: 모바일에서도 이용할 수 있나요?

**A:** 네, 모든 기기(PC, 태블릿, 스마트폰)에서 이용 가능합니다.
모바일 홈 화면에 추가하면 앱처럼 사용할 수 있습니다.

---

### Q: 한국어 외에 다른 언어도 지원하나요?

**A:** 네, 한국어와 영어를 지원합니다. 화면 상단의 언어 설정에서 변경할 수 있습니다.

---

## 2. 회원가입/로그인

### Q: 어떻게 회원가입하나요?

**A:** 세 가지 방법으로 가입할 수 있습니다:

1. **이메일 가입**: 이메일과 비밀번호 입력
2. **GitHub 로그인**: GitHub 계정으로 원클릭 가입
3. **Google 로그인**: Google 계정으로 원클릭 가입

---

### Q: 비밀번호를 잊어버렸어요.

**A:** 비밀번호 찾기 기능을 이용해 주세요:

1. 로그인 페이지 접속
2. "비밀번호 찾기" 클릭
3. 가입한 이메일 입력
4. 이메일로 전송된 링크로 비밀번호 재설정

---

### Q: 소셜 로그인(GitHub/Google)을 사용하면 비밀번호가 없나요?

**A:** 네, 소셜 로그인은 해당 서비스의 인증을 사용합니다.
일반 비밀번호로 로그인하려면 "비밀번호 설정"에서 새 비밀번호를 만드세요.

---

### Q: 이메일 인증 메일이 안 와요.

**A:** 다음을 확인해 주세요:

1. 스팸/정크 메일함 확인
2. 이메일 주소 오타 확인
3. 인증 메일 재발송 요청
4. 계속 안 되면 support@vibeolympics.com으로 문의

---

## 3. 상품 구매

### Q: 상품은 어떻게 구매하나요?

**A:** 간단한 4단계로 구매할 수 있습니다:

1. **판도라 샵**에서 원하는 상품 선택
2. **"구매하기"** 버튼 클릭
3. **결제 수단** 선택 및 결제
4. **즉시 다운로드** 가능

---

### Q: 구매 전에 상품 내용을 미리 볼 수 있나요?

**A:** 네, 대부분의 상품에서 다음을 확인할 수 있습니다:

- 상품 설명 및 기능 목록
- 미리보기 이미지/영상
- 샘플 파일 (판매자 제공 시)
- 다른 구매자의 리뷰

---

### Q: 위시리스트는 어떻게 사용하나요?

**A:** 마음에 드는 상품을 저장해 둘 수 있습니다:

1. 상품의 하트 아이콘 클릭
2. **대시보드 > 위시리스트**에서 확인
3. 할인이나 업데이트 시 알림 받기 가능

---

### Q: 구매 영수증은 어디서 받나요?

**A:** 두 가지 방법으로 확인할 수 있습니다:

- **이메일**: 결제 완료 시 자동 발송
- **대시보드 > 구매 내역**: 언제든 다시 확인/출력 가능

---

## 4. 결제

### Q: 어떤 결제 수단을 사용할 수 있나요?

**A:** 다양한 결제 수단을 지원합니다:

**국내 결제**
| 결제수단 | 지원 |
|----------|------|
| 신용/체크카드 | ✅ 모든 국내 카드 |
| 카카오페이 | ✅ |
| 토스페이 | ✅ |
| 네이버페이 | ✅ |
| 계좌이체 | ✅ 실시간 |
| 가상계좌 | ✅ 무통장입금 |

**해외 결제**
| 결제수단 | 지원 |
|----------|------|
| PayPal | ✅ |
| 해외카드 (Visa, Mastercard 등) | ✅ Stripe |

---

### Q: 결제가 실패해요.

**A:** 다음 사항을 확인해 주세요:

1. **카드 한도/잔액** 확인
2. **해외결제 차단** 여부 확인 (일부 카드)
3. **다른 결제 수단**으로 시도
4. **팝업 차단** 해제 확인

계속 안 되면 support@vibeolympics.com으로 연락해 주세요.
(오류 화면 캡처 첨부 시 더 빠른 해결 가능)

---

### Q: 결제 후 확인 메시지가 안 와요.

**A:** 다음을 확인해 주세요:

1. **스팸 메일함** 확인
2. **대시보드 > 구매 내역**에서 직접 확인
3. 결제가 완료되었다면 상품 이용에는 문제없음
4. 확인이 안 되면 고객센터 문의

---

### Q: 중복 결제된 것 같아요.

**A:** 즉시 고객센터로 연락해 주세요:

- 이메일: support@vibeolympics.com
- 제목에 **[긴급] 중복 결제**라고 적어주세요
- 결제 시간, 금액, 결제 수단 정보 포함

확인 후 중복분은 **전액 환불** 처리해 드립니다.

---

## 5. 다운로드/이용

### Q: 구매한 상품은 어디서 다운로드하나요?

**A:** **대시보드 > 구매 내역**에서 다운로드할 수 있습니다.

- 구매한 모든 상품 목록 확인 가능
- **언제든 다시 다운로드 가능** (횟수 제한 없음)
- 상품 업데이트 시 최신 버전도 다운로드 가능

---

### Q: 다운로드가 안 돼요.

**A:** 다음을 시도해 주세요:

1. **다른 브라우저**로 시도 (Chrome 권장)
2. **팝업 차단 해제**
3. **네트워크 연결** 확인
4. **대용량 파일**의 경우 Wi-Fi 환경 권장

계속 안 되면 support@vibeolympics.com으로 문의해 주세요.

---

### Q: 라이선스는 어떻게 되나요?

**A:** 각 상품 페이지에 라이선스가 명시되어 있습니다:

| 라이선스 | 개인 사용 | 상업적 사용 | 재판매 |
|----------|----------|------------|--------|
| 개인용 | ✅ | ❌ | ❌ |
| 상업용 | ✅ | ✅ | ❌ |
| 확장 | ✅ | ✅ (팀/기업) | ❌ |

**공통**: 원본 파일 재판매는 모든 라이선스에서 금지됩니다.

---

### Q: 구매한 상품으로 만든 결과물을 판매해도 되나요?

**A:** 라이선스에 따라 다릅니다:

- **개인용**: 개인 프로젝트에만 사용 가능
- **상업용**: 상업적 프로젝트에 활용 가능
- **공통**: 원본 상품 자체를 재판매하는 것은 금지

자세한 내용은 각 상품의 라이선스 조항을 확인해 주세요.

---

## 6. 판매자

### Q: 누구나 판매자가 될 수 있나요?

**A:** 네! 별도의 신청 절차 없이 **바로 판매를 시작**할 수 있습니다.

- 회원가입만 하면 누구나 판매 가능
- 복잡한 인증 절차 없음
- 상품 등록 즉시 판매 시작

---

### Q: 상품 등록은 어떻게 하나요?

**A:** 간단한 절차로 등록할 수 있습니다:

1. **로그인** 후 **대시보드** 접속
2. **내 상품 > 새 상품 등록** 클릭
3. 상품 정보 입력 (제목, 설명, 가격, 파일)
4. **등록** 버튼 클릭 → 즉시 판매 시작

---

### Q: 어떤 상품을 팔 수 있나요?

**A:** 다양한 디지털 상품을 판매할 수 있습니다:

- **디지털 상품**: 웹앱, 템플릿, 코드, 도구
- **AI 도서**: 전자책, 만화, 동화
- **AI 영상**: 애니메이션, 단편 영화
- **AI 음악**: BGM, 앰비언트, 로열티프리

**판매 금지 상품**: 불법 콘텐츠, 저작권 침해물, 악성코드

---

### Q: 무료 상품도 등록할 수 있나요?

**A:** 네! 무료 상품도 등록 가능합니다.
포트폴리오 용도나 샘플 제공 등에 활용하세요.

---

### Q: 가격은 어떻게 설정하나요?

**A:** 자유롭게 설정할 수 있습니다:

- **최소 가격**: 1,000원
- **최대 가격**: 제한 없음
- **AI 가격 추천**: 유사 상품 기반 추천 가격 제공

---

### Q: 수수료는 얼마인가요?

**A:** 판매 시 다음 수수료가 적용됩니다:

| 항목 | 비율 | 설명 |
|------|------|------|
| 플랫폼 수수료 | 10% | Vibe Olympics 서비스 이용료 |
| PG 수수료 | 약 2.5~3.5% | 결제 대행 수수료 (수단별 상이) |

**예시**: 10,000원 상품 판매 시
→ 플랫폼 1,000원 + PG 약 300원 = 약 8,700원 수령

## 7. 정산

### Q: 정산은 언제 받나요?

**A:** 정산 일정:

| 항목 | 내용 |
|------|------|
| 정산일 | 매월 1일 (전월 판매분) |
| 최소 금액 | 10,000원 이상 |
| 입금 시점 | 정산일로부터 3~5영업일 |

---

### Q: 정산 계좌는 어떻게 등록하나요?

**A:** **대시보드 > 설정 > 정산 정보**에서 등록할 수 있습니다:

1. 은행 선택
2. 계좌번호 입력
3. 예금주 확인
4. 저장

---

### Q: 최소 정산 금액에 못 미치면 어떻게 되나요?

**A:** 다음 달로 이월됩니다.

- 10,000원 미만 수익: 다음 정산 시 합산
- 누적 금액이 10,000원 이상 되면 정산

---

### Q: 정산 내역은 어디서 확인하나요?

**A:** **대시보드 > 정산 내역**에서 확인할 수 있습니다:

- 판매 내역별 상세 정보
- 수수료 차감 내역
- 입금 완료 내역
- 정산 예정 금액

---

## 8. 환불

### Q: 환불이 가능한 경우는 언제인가요?

**A:** 다음 경우 환불이 가능합니다:

| 사유 | 환불 가능 | 비고 |
|------|----------|------|
| 상품 설명과 현저히 다름 | ✅ | 증거 필요 |
| 파일 손상/다운로드 불가 | ✅ | 기술팀 확인 |
| 중복 결제 | ✅ | 즉시 처리 |
| 구매 후 7일 이내 (다운로드 전) | ✅ | |
| 다운로드 후 단순 변심 | ❌ | 디지털 상품 특성 |
| 구매자 환경 문제 (PC 사양 등) | ❌ | |

---

### Q: 환불 요청은 어떻게 하나요?

**A:** **대시보드 > 구매 내역**에서 요청할 수 있습니다:

1. 환불할 상품 선택
2. **"환불 요청"** 클릭
3. 환불 사유 선택 및 상세 내용 작성
4. 증거 자료 첨부 (해당 시)
5. 제출

---

### Q: 환불 처리 기간은 얼마나 걸리나요?

**A:** 전체 약 4~10일 정도 소요됩니다:

| 단계 | 기간 |
|------|------|
| 환불 심사 | 1~3 영업일 |
| 환불 승인 후 처리 | 3~7 영업일 (결제 수단별) |

---

### Q: 부분 환불도 가능한가요?

**A:** 상황에 따라 가능합니다:

- 분쟁 조정 과정에서 합의된 경우
- 상품의 일부만 문제가 있는 경우
- 운영팀 판단에 따른 조정

---

## 9. 신고/분쟁

### Q: 문제가 있는 상품/사용자를 신고하려면?

**A:** 두 가지 방법으로 신고할 수 있습니다:

1. **상품/리뷰/게시글 페이지**: 신고 버튼 클릭
2. **고객센터**: support@vibeolympics.com으로 상세 내용 전송

신고 시 **증거 자료**(스크린샷 등)를 첨부하면 더 빠른 처리가 가능합니다.

---

### Q: 어떤 경우에 신고할 수 있나요?

**A:** 다음 경우 신고할 수 있습니다:

| 유형 | 예시 |
|------|------|
| 사기/기만 | 허위 상품, 돈만 받고 상품 미제공 |
| 저작권 침해 | 무단 복제, 도용 |
| 부적절한 콘텐츠 | 불법, 유해, 성인물 |
| 괴롭힘/욕설 | 비방, 협박, 모욕 |
| 직거래 유도 | 플랫폼 밖 거래 시도 |
| 품질 문제 | 설명과 다른 상품 |

---

### Q: 신고하면 어떻게 처리되나요?

**A:** 다음 절차로 처리됩니다:

1. **접수** (24시간 내): 신고 내용 검토
2. **피신고인 통지**: 소명 기회 부여 (48시간)
3. **사실 확인**: 증거 검토
4. **결정 및 통보**: 결과 안내

처리 기간: 일반적으로 3~7영업일

---

### Q: 거래 분쟁이 생기면 어떻게 하나요?

**A:** 분쟁 해결은 3단계로 진행됩니다:

**1단계: 당사자 협의 (48시간)**
- 판매자와 직접 대화로 해결 시도
- 분쟁 채팅방에서 진행

**2단계: 운영팀 중재 (72시간)**
- 협의 실패 시 운영팀이 개입
- 양측 주장 검토 후 중재안 제시

**3단계: 최종 판정 (48시간)**
- 중재 거부 시 최종 판정
- 판정 결과에 따라 환불/정산 처리

---

### Q: 분쟁 신청은 어떻게 하나요?

**A:** **대시보드 > 구매 내역**에서 신청할 수 있습니다:

1. 해당 거래 선택
2. **"분쟁 신청"** 클릭
3. 분쟁 유형 및 사유 작성
4. 증거 자료 첨부
5. 제출

**신청 조건**: 거래일로부터 14일 이내

---

### Q: 허위 신고를 당하면 어떻게 되나요?

**A:** 걱정하지 않으셔도 됩니다:

- 모든 신고는 **사실 확인 후** 처리
- **소명 기회**가 반드시 주어짐
- 증거 없이 일방적 제재하지 않음
- 악의적 허위 신고자에게는 **역제재** 적용

---

## 10. 계정 보안

### Q: 비밀번호 변경은 어떻게 하나요?

**A:** **대시보드 > 설정 > 비밀번호 변경**에서 할 수 있습니다:

1. 현재 비밀번호 입력
2. 새 비밀번호 입력
3. 새 비밀번호 확인
4. 저장

---

### Q: 계정이 해킹된 것 같아요.

**A:** 즉시 다음 조치를 취해 주세요:

1. **비밀번호 즉시 변경**
2. **로그인 기록 확인** (설정 > 보안)
3. **의심스러운 활동 신고**: support@vibeolympics.com
4. **연결된 소셜 계정**도 비밀번호 변경

---

### Q: 계정을 삭제하려면 어떻게 하나요?

**A:** **대시보드 > 설정 > 계정 삭제**에서 가능합니다.

⚠️ **주의사항**:
- 삭제 시 **모든 데이터 영구 삭제**
- 구매 내역, 판매 상품, 리뷰 등 복구 불가
- 미정산금은 삭제 전 정산 필요

---

### Q: 개인정보는 안전하게 보호되나요?

**A:** 네, 다음과 같이 보호합니다:

- **암호화**: 모든 개인정보 암호화 저장
- **보안 인증**: SSL 인증서 적용
- **접근 제한**: 필요 최소한의 인원만 접근
- **개인정보처리방침**: /privacy 페이지 참고

---

## 11. 기술 지원

### Q: 사이트가 느리거나 오류가 나요.

**A:** 다음을 시도해 주세요:

1. **브라우저 새로고침** (Ctrl+F5)
2. **캐시/쿠키 삭제**
3. **다른 브라우저**로 시도 (Chrome 권장)
4. **네트워크 연결** 확인

계속 문제가 있으면 support@vibeolympics.com으로 문의해 주세요.
(오류 화면, 브라우저 정보 첨부 시 더 빠른 해결 가능)

---

### Q: 파일이 열리지 않아요.

**A:** 다음을 확인해 주세요:

1. **파일 형식**에 맞는 프로그램 설치 여부
2. 상품 페이지의 **시스템 요구사항** 확인
3. **파일 압축 해제** 필요 여부 (.zip 파일)
4. 재다운로드 시도

---

### Q: 지원하는 브라우저는 무엇인가요?
